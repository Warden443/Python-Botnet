### Uso
1. Introduzca la IP de la víctima.
2. Ingrese el puerto.
3. Elija su tipo de ataque.
4. Seleccione su método de ataque.
5. Elija el número de hilos.
6. Presiona "Enter" para iniciar el ataque.
